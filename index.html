ServoControl

This is the Servo Control program for the servo board produced by WE Bots.

Status

Please note that this is a work in progress! ServoControl is a developmental project - the master branch may break or be in a broken state at anytime.

Setup

This project is built using the development environment Qt Creator. (You need to have Qt installed before compiling and working on this.)
